# Product Overview

**Análise de Loterias** is a statistical analysis system for Brazilian lottery games (Lotofácil and Mega-Sena).

## Purpose

Educational tool for studying programming and data analysis through lottery statistics. The application analyzes historical draw data from Excel files and generates number combinations using various statistical strategies.

## Key Features

- Statistical analysis dashboard showing frequency patterns, hot/cold numbers, and distribution metrics
- Multiple generation strategies: smart mix, weighted frequency, pattern-based, and pure random
- Support for two lottery types:
  - **Lotofácil**: 15 numbers from 1-25
  - **Mega-Sena**: 6 numbers from 1-60
- Interactive UI with game switching, customizable generation parameters, and exportable results

## Important Notes

- This is an educational project for studying statistics and programming
- Results are based on historical data analysis but lottery outcomes are random
- Users should play responsibly
- Content and UI are in Brazilian Portuguese (pt-BR)
